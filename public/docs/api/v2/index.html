EventaServo API v2

EventaServo is a platform for organizing and publicizing Esperanto events worldwide. This API v2 provides programmatic access to events and organizations.

Events

Event-related endpoints

List events

List events by date range, country, or organization. You must provide either komenca_dato + fina_dato or uuid.

Authorizations:
JWTToken
query Parameters
komenca_dato
string <date>
Example: komenca_dato=2026-01-01

Start date of the filter period (YYYY-MM-DD). Required when uuid is not provided.

fina_dato
string <date>
Example: fina_dato=2026-12-31

End date of the filter period (YYYY-MM-DD). Required when uuid is not provided.

uuid
string <uuid>
Example: uuid=e567bdc3-0ea2-46cc-8a83-3b11facbe16c

UUID of a specific event. When provided, komenca_dato and fina_dato are not needed.

landa_kodo
string
Example: landa_kodo=BR

Filter by 2-letter country code. Case-insensitive (e.g. BR, br, Fr).

o
string
Example: o=UEA

Filter by organization short name (e.g. UEA).

Responses

Production

https://eventaservo.org/api/v2/eventoj

Organizations

Organization-related endpoints

List organizations

List all organizations, optionally filtered by country code.

Authorizations:
JWTToken
query Parameters
country_code
string
Example: country_code=BR

Filter by 2-letter country code (e.g. BR, NL).
